# Conversao-voz-para-chatgpt-com-whisper-python
Este projeto grava áudio do usuário, converte em texto utilizando o Whisper e envia o resultado para um modelo de IA gerar uma resposta. Demonstra um pipeline completo de entrada por voz, transcrição e interação com IA usando Python, com foco na integração prática de tecnologias modernas.
